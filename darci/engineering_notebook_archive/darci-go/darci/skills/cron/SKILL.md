# Cron Skill

Scheduled task management for the darci agent.

## Description

This skill enables the darci agent to create, manage, and execute scheduled tasks using cron expressions.

## Tools

- `cron_add`: Add a scheduled task
- `cron_list`: List all scheduled tasks
- `cron_remove`: Remove a scheduled task
- `cron_run`: Run a scheduled task immediately

## Usage

Ask the agent to:
- "Schedule a daily reminder at 9 AM"
- "List all my scheduled tasks"
- "Remove the hourly check task"
