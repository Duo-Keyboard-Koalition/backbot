# Weather Skill

Weather information for the darci agent.

## Description

This skill enables the darci agent to fetch and provide weather information for any location.

## Tools

- `weather_current`: Get current weather for a location
- `weather_forecast`: Get weather forecast for a location
- `weather_alerts`: Get weather alerts for a location

## Usage

Ask the agent to:
- "What's the weather like in New York?"
- "Give me the 7-day forecast for London"
- "Are there any weather alerts for Tokyo?"
