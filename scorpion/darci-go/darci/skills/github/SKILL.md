# GitHub Skill

GitHub integration for the darci agent.

## Description

This skill enables the darci agent to interact with GitHub repositories, issues, pull requests, and more.

## Tools

- `github_search_repos`: Search for GitHub repositories
- `github_get_repo`: Get repository information
- `github_list_issues`: List issues in a repository
- `github_create_issue`: Create a new issue
- `github_get_pull_request`: Get pull request details
- `github_list_pull_requests`: List pull requests

## Usage

Ask the agent to:
- "Search for darci projects on GitHub"
- "List open issues in HKUDS/darci"
- "Create an issue for bug fix"
