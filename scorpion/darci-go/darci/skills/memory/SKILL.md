# Memory Skill

Memory management for the darci agent.

## Description

This skill enables the darci agent to manage and query its memory, including short-term and long-term memory storage.

## Tools

- `memory_add`: Add a memory entry
- `memory_search`: Search memories by keyword
- `memory_list`: List recent memories
- `memory_delete`: Delete a memory
- `memory_clear`: Clear all memories

## Usage

Ask the agent to:
- "Remember that my favorite language is Go"
- "What do you remember about my projects?"
- "Search memories about Python"
