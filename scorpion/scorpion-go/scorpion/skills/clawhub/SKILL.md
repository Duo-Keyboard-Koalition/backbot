# ClawHub Skill

Search and install public agent skills from ClawHub.

## Description

This skill enables the scorpion agent to search for and install skills from the ClawHub marketplace.

## Tools

- `clawhub_search`: Search for skills in ClawHub
- `clawhub_install`: Install a skill from ClawHub
- `clawhub_list`: List installed skills

## Usage

Ask the agent to search for or install skills:
- "Search for weather skills on ClawHub"
- "Install the github skill from ClawHub"
