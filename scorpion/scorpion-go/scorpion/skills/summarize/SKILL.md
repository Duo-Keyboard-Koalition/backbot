# Summarize Skill

Text summarization for the scorpion agent.

## Description

This skill enables the scorpion agent to summarize long texts, articles, and documents.

## Tools

- `summarize_text`: Summarize a given text
- `summarize_url`: Fetch and summarize a web article
- `summarize_file`: Summarize a file's contents

## Usage

Ask the agent to:
- "Summarize this article: [URL]"
- "Give me a summary of the document"
- "Create a brief summary of this text"
